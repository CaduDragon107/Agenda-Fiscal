| client form → Server Action | disabled selects are UX only; real authority is the server-side guard (Plan 03) |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-15 | Information Disclosure | colaborador's rendered table includes other sectors' responsável data (hidden column still ships in payload) | mitigate | For non-DONO, build ONLY the user's own-sector column in the columns array; do not include other sectors' responsável values in `EmpresaRow` rows sent to a colaborador (page.tsx derives per-sector data but the table renders only the user's sector for non-DONO) — verified in human checkpoint step 3 |
